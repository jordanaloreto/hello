🔐 Hello

Este projeto é uma adaptação do desafio:

👉 Hello!

Com um foco adicional em segurança de aplicações (DevSec).

🎯 Objetivo

Demonstrar, de forma prática, quais informações podem ser obtidas a partir de um dado simples como o endereço IP, e conscientizar sobre os riscos envolvidos.

🧠 Mentalidade do Projeto

Este projeto faz parte de uma jornada para a área de DevSecOps, com foco em:

Construir aplicações seguras desde a base

Entender como ataques funcionam na prática

Aplicar contramedidas diretamente no código

“Como isso pode ser explorado?”

“Como prevenir isso desde o início?”

🚀 Motivação

Pequenas decisões no código têm grande impacto na segurança.

Algumas linhas bem implementadas podem ser a diferença entre:

🔥 Apagar incêndios em produção

👀 Apenas monitorar um sistema seguro

🔐 Boas práticas de navegação

Baseado em recomendações da Fortinet:

🔑 Use senhas fortes + 2FA

🛜 Evite Wi-Fi público (ou use VPN)

🧠 Cuidado com phishing

🔄 Mantenha sistemas atualizados

🔒 Verifique HTTPS antes de inserir dados

🚫 Não compartilhe informações desnecessárias

🌐 Uilize navegadores com foco na privacidade como o Brave
⚠️ Aviso

Este projeto é educacional.
Nenhum dado é armazenado — apenas exibido para demonstrar o que pode ser coletado automaticamente durante a navegação.
